# Unity Game
This is a Unity prototype developed as a university project. It is a small RPG adventure game where the player must eliminate enemies to reach the final boss, the Lich King, and defeat him.

The game features various enemies with distinct attack patterns (most have at least two), as well as weapons with unique effects and gameplay abilities. The GIFs below demonstrate the core gameplay features.
